# 📚 JNVU BCA IV Semester Study Planner

A comprehensive interactive study planner for JNVU (Jai Narain Vyas University) BCA IV Semester examination with detailed subject guides, exam tips, and a complete 7-day study schedule.

## 📖 Features

- **📅 Study Plan**: 7-day study schedule covering all 4 subjects with daily topics and time allocations
- **📖 Subject Guide**: Detailed unit-wise breakdowns for:
  - CSA 6005T - Introduction to Internet Technology
  - CSA 6006T - Operating System
  - CSA 6007T - Introduction to Python Programming
  - CSA 6008T - Client Server Computing
- **💡 Exam Tips**: Strategies for exam success including:
  - 12-mark answer formatting
  - 2-mark question strategy
  - Numerical problem solving tips
  - Python programming approach
  - Time management techniques
  - Pre-exam preparation checklist

## 🔥 Key Highlights

- **12-Mark Answer Blueprints** for every unit with detailed explanations
- **2-Mark Questions** - Expected questions for each unit
- **Complete Exam Information** with dates, times, and subject codes
- **Interactive UI** with expandable unit cards
- **Responsive Design** works on desktop, tablet, and mobile

## 🚀 Live Demo

Open `index.html` in any modern web browser - no installation needed!

## 📱 How to Use

1. Click the **Study Plan** tab to see your 7-day schedule
2. Browse the **Subject Guide** tab to explore detailed unit content
3. Read the **Exam Tips** for strategic advice
4. Click on any unit card to expand and see:
   - 12-mark answer blueprint
   - Expected 2-mark questions

## ⚡ Quick Tips

- Focus on **🔥 12-mark topics** first (they're worth 60% of each exam)
- Practice numerical problems from the OS paper daily
- Write complete Python programs (not just snippets)
- Review exam tips the night before each exam
- Get 7-8 hours of sleep before each exam!
